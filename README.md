# 📊 PhonePe Pulse Data Visualization 
Interactive Power BI dashboard analyzing PhonePe Pulse dataset (2018–2022). Covers 71.7bn transactions worth ₹121T across India — state-wise geographic analysis, user behavior, device trends &amp; transaction category insights. Built with 4 dashboards using maps, slicers, combo charts &amp; KPI cards.

## 📌 Problem Statement
PhonePe has a large amount of raw transaction and user data which is difficult to understand directly.  
The main challenge was to convert this raw data into meaningful insights using interactive dashboards and geo visualizations.

---

## ⚠️ Problems Faced
- Large raw datasets with multiple CSV files
- Difficult to identify transaction trends directly
- Handling missing values and inconsistent data
- Creating interactive geo visualizations
- Understanding regional performance and user growth

---

## 🛠️ Solutions Implemented
- Cleaned and organized the datasets
- Removed duplicates and null values
- Standardized state and district names
- Created KPI cards, maps, and charts
- Added slicers for better interactivity and filtering
- Visualized trends using line charts and geo maps
